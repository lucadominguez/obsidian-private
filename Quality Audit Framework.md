# Content Quality Audit Framework

Use this rubric for every article, infographic, and playbook before publication.

## Accuracy Audit (Pass/Fail)

| Dimension | Check | Required |
|---|---|---|
| **Numbers** | Every specific number (dosing, percentage, dollar amount) verified against source | ✅ Must pass |
| **Citations** | Every paper/journal reference has a real DOI or PMID | ✅ Must pass |
| **Claims** | No unsupported superlatives ("best," "only," "always") without evidence | ✅ Must pass |
| **Units** | All lab values have correct units (mg/L vs µg/dL, ng/mL vs pg/mL) | ✅ Must pass |
| **Mechanisms** | Biological/chemical mechanisms accurately described | ✅ Must pass |
| **Contradictions** | No internal contradictions within or across articles | ✅ Must pass |
| **Disclaimers** | Medical/legal/financial disclaimer present where needed | ✅ Must pass |

## Coherence Audit (Score 1-5)

| Dimension | What to check |
|---|---|
| **Narrative arc** | Does the article build from hook → evidence → insight → action? |
| **Section flow** | Do transitions make sense? No jarring topic switches? |
| **Voice consistency** | Does the persona voice stay consistent throughout? |
| **Audience fit** | Appropriate technical depth for target reader? |
| **Call to action** | Clear next step? Lead magnet relevant to content? |

## Visual Audit (Score 1-5)

| Dimension | What to check |
|---|---|
| **Data accuracy** | Visual representation matches numbers in text |
| **Readability** | Labels legible, axes clear, colors distinguishable |
| **Information density** | Neither sparse nor cluttered |
| **Professional quality** | Would this pass peer review? Would a paying subscriber find it valuable? |
| **Source attribution** | Figures sourced from published papers cited properly |

- **Pre-publication**: Full audit before any article goes live
